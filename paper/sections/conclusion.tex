\section{Conclusion}
\label{sec:conclusion}

We have presented \textsc{TaxonomyAgent}, an open-source
orchestrator--judge framework for discovering structured taxonomies
in unstructured text, designed for analyst workflows where the
categories are not known a priori. A one-sentence goal instruction
steers discovery toward a chosen axis of variation, which makes the
system applicable both to safety-relevant auditing (such as
cataloguing the manipulation patterns in a corpus of model
interactions) and to classical taxonomy induction. At the install
default (DeepSeek-v4-Flash for both roles), \textsc{TaxonomyAgent}
recovers the six-category expert taxonomy of
DarkBench~\citep{kran2025darkbench} at NMI $0.927 \pm 0.018$, ahead of
every baseline asked to group by the same manipulation type, and
reaches NMI
$0.740 \pm 0.055$ on 20~Newsgroups, beating a published
topic-discovery pipeline by $1.22\times$ on the classical benchmark.
We release the framework with a one-line \texttt{pip install} path
and a Streamlit Inspect tab that streams the trace, partial
taxonomies, and cost split of any run; code and reproduction
scripts are at \repourl.
