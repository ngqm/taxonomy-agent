\section{Conclusion}
\label{sec:conclusion}

We present an orchestrator/judge agent whose primary contribution is the operational layer that lets a stranger run it at a booth: live provider-native cost accounting, partial-progress streaming, vendor-agnostic routing, and a Streamlit Inspect tab over a typed dispatcher and a parallel stateless judge. At the install default (DeepSeek-v4-Flash for both roles), the system reaches NMI $0.713$ on 20~Newsgroups at USD $0.16$ per run, beating a model-matched embed-cluster-LLM reimplementation of TnT-LLM by $1.5\times$, and labels 132 of 149 items at $100\%$ accuracy on a 149-item synthetic CoT-failure corpus. A Sonnet 4.6 orchestrator pair at twelve times the cost sits inside the cheap pair's NMI interval on 20NG, so the cheap pair is the publication and install default.
