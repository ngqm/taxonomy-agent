\section{Conclusion}
\label{sec:conclusion}

Taxonomy induction tools rarely survive the trip from research prototype to something a stranger can run at a booth. We present an open-source orchestrator/judge agent whose primary contribution is the operational layer that closes that gap: live native provider cost accounting split by orchestrator and judge, partial-progress streaming so an interrupted run leaves recoverable state, vendor-agnostic routing, and a browser UI for live inspection. Beneath sit a typed dispatcher that validates each edit before mutation and a parallel stateless judge. At the install default, DeepSeek-v4-Flash for both roles, the system reaches NMI $0.713$ and ARI $0.452$ on 20~Newsgroups at USD $0.16$ per run, beating a one-shot LLM baseline by $2.1\times$ on NMI and a model-matched embed-cluster-LLM reimplementation of TnT-LLM by $1.5\times$ on NMI. On a 149-item synthetic corpus of chain-of-thought failures it labels 132 of 149 items at $100\%$ accuracy and reaches $0.886$ coverage-adjusted purity, $+16$ points above a supervised surface oracle. A Sonnet 4.6 orchestrator pair at twelve times the cost lands inside the cheap pair's NMI interval on 20NG and under-recovers \texttt{hallucinated\_premise} on CoT, so the cheap pair is the publication and install default. Taxonomy induction is now operable from a demo booth at install-default cost alongside the results table.
