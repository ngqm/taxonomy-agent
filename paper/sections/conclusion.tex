\section{Conclusion}
\label{sec:conclusion}

We have presented \textsc{TaxonomyAgent}, an open-source
orchestrator--judge framework for discovering structured taxonomies
in unstructured text, designed for analyst workflows where the
categories are not known a priori. A one-sentence goal instruction
steers discovery toward a chosen axis of variation, which makes the
system applicable both to underserved tasks in LLM alignment
research (surfacing CoT failure modes, reasoning strategies, error
types) and to classical taxonomy-induction tasks. At the install
default (DeepSeek-v4-Flash for both roles), \textsc{TaxonomyAgent}
reaches NMI $0.944 \pm 0.081$ on a synthetic CoT-failure corpus
where the published TopicGPT pipeline collapses to a single surface
topic, and NMI $0.740 \pm 0.055$ on 20~Newsgroups, beating TopicGPT
by $1.22\times$ on the classical benchmark. A Sonnet 4.6 orchestrator
at $11\times$ the cost does not improve over the cheap pair on
20NG, so orchestrator quality is not the bottleneck at this scale.
We release the framework with a one-line \texttt{pip install} path
and a Streamlit Inspect tab that streams the trace, partial
taxonomies, and cost split of any run; code and reproduction
scripts are at \repourl.
