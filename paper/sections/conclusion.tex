\section{Conclusion}
\label{sec:conclusion}

We present an orchestrator/judge agent built on two structural commitments: a typed dispatcher with validate-before-mutate and per-operation failure isolation, and a stateless judge that re-derives each item's category from item text alone. At the install default (DeepSeek-v4-Flash for both roles), the system reaches NMI $0.727$ on 20~Newsgroups at USD $0.17$ per run, $1.5\times$ a model-matched embed-cluster-LLM reimplementation of TnT-LLM, and reaches NMI $0.944$ across three seeds on a 149-item synthetic CoT-failure corpus. A Sonnet 4.6 orchestrator at $11\times$ the cost lands inside the cheap pair's interval, so orchestrator quality is not the bottleneck.
